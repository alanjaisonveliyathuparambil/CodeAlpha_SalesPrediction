# Sales Prediction using Python

## Project Overview

This project predicts product sales based on advertising expenditure on TV, Radio, and Newspaper platforms. Machine Learning techniques were used to analyze advertising impact and build a predictive model.

## Objectives

* Analyze advertising data.
* Explore relationships between advertising channels and sales.
* Build a sales prediction model using Linear Regression.
* Evaluate model performance using standard metrics.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

* Advertising.csv

## Visualizations

1. TV Advertising vs Sales
2. Correlation Heatmap
3. Actual vs Predicted Sales

## Model Used

Linear Regression

## Model Performance

* MAE: 1.46
* MSE: 3.17
* R² Score: 0.899

## Key Findings

* TV advertising has the strongest influence on sales.
* Radio advertising has a moderate impact on sales.
* Newspaper advertising has the weakest influence on sales.
* The Linear Regression model achieved nearly 90% prediction accuracy (R² Score = 0.899).

## Conclusion

This project successfully analyzed advertising data and predicted sales using Machine Learning. The model demonstrated strong predictive performance and showed that advertising expenditure, especially on TV, plays a significant role in driving sales.
